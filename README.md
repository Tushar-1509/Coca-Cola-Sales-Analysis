# Coca Cola Sales Dashboard

## Project Overview

Developed an interactive sales dashboard to analyze beverage sales performance across different regions, retailers, and product brands.

---

## Dashboard Features

- Total Sales
- Units Sold
- Average Operating Profit
- Monthly Sales Trends
- Regional Sales Analysis
- State-wise Sales Map
- Interactive Slicers

---

## Tools Used

- Microsoft Excel

---

## Skills Demonstrated

- Dashboard Development
- KPI Reporting
- Pivot Tables
- Interactive Reporting
- Business Analytics

---

## Business Questions Answered

- Which regions perform best?
- Which retailers generate the highest sales?
- Which beverage brands perform better?
